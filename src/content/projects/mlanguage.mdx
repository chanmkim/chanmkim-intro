# 포트폴리오 웹사이트

개인 포트폴리오를 위한 모던하고 반응형 웹사이트를 개발했습니다. Next.js 14의 최신 기능들을 활용하여 빠르고 SEO 친화적인 사이트를 구축했습니다.

## 기술 스택

- **Frontend**: Next.js 14, TypeScript, React 18
- **Styling**: Tailwind CSS, CSS Modules
- **Content**: MDX (Markdown + JSX)
- **Deployment**: Vercel
- **Version Control**: Git, GitHub

## 개발 과정

### 초기 기획 단계
프로젝트 시작 전에 다음과 같은 요구사항을 정의했습니다:
- 개인 포트폴리오 및 블로그 통합
- 빠른 로딩 속도와 SEO 최적화
- 모던한 UI/UX 디자인

### 기술 선택
Next.js 14를 선택한 이유:
- App Router의 Server Components 활용
- 자동 코드 분할 및 최적화
- Vercel과의 완벽한 호환성

### 개발 중 겪은 어려움
1. **Server Components vs Client Components**
   - 처음에는 모든 컴포넌트를 Server Component로 만들려 했음
   - 인터랙티브 요소가 필요한 부분은 Client Component로 분리해야 함을 깨달음

2. **MDX 설정**
   - `next-mdx-remote` 패키지 설정 과정에서 어려움
   - 프론트매터 데이터 처리 방법 학습

3. **반응형 디자인**
   - Tailwind CSS의 브레이크포인트 활용법 숙지
   - 모바일 우선 접근법 적용

## 결과 및 배운 점

### 성과
- **성능**: Lighthouse 점수 95점 이상 달성

- **SEO**: Google Search Console에서 좋은 인덱싱 결과

### 배운 점
- **Next.js 14의 새로운 패러다임**: Server Components의 장점과 한계점 이해
